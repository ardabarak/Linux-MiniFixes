###permanently 0->1 turbo_boost status at: "/sys/devices/system/cpu/intel_pstate/"
###disabling turbo boost permanently at boot


###Create a service file at /etc/systemd/system/disable-turbo-boost.service
>>>     sudo systemctl enable disable-turbo-boost.service

```
[Unit]
Description=Disable Turbo Boost on Intel CPU

[Service]
ExecStart=/bin/sh -c "/usr/bin/echo 1 > /sys/devices/system/cpu/intel_pstate/no_turbo"
ExecStop=/bin/sh -c "/usr/bin/echo 0 > /sys/devices/system/cpu/intel_pstate/no_turbo"
RemainAfterExit=yes

[Install]
WantedBy=sysinit.target
```

###After saving the file, enable the service to start on boot:
>>>     sudo systemctl enable disable-turbo-boost.service
